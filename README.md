Le projet a été  écrit, compil é et testé avec le SDK de Java version 16. Un fichier .jar est fourni, permettant de
lancer simplement le programme. Pour l?utiliser il faut se placer dans le dossier racine et taper la commande :
`` java -jar exe/tp5.jar ``

Si toutefois vous souhaitez compiler le projet vous meme, vous pouvez en vous pla çant dans le dossier racine et en
tapant la commande :
`` javac ./**/*.java -encoding ISO-8859-1 -d out/production ``
et enfin vous pouvez exécuter le programme en tapant la commande suivante : `` java -Dfile.encoding=ISO-8859-1 -classpath out/production src.Main ``
